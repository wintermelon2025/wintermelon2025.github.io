数据说明

本界面提供的查询服务数据均来自公开表格，并经过系统化对照处理。由于数据来源的复杂性，若发现重名或数据错误，敬请谅解。

如需提交错误信息，请访问：错误反馈入口

技术说明

本系统采用纯前端技术（HTML/CSS/JavaScript）开发，并通过 GitHub Pages 提供服务。因 GitHub Pages 的特性，偶尔可能出现链接访问延迟或不稳定的情况，建议稍后重试。

此外，查询服务中涉及的首字母生成功能依赖于 Python 的第三方库 pypinyin。该库可以将汉字转换为拼音，并提取首字母，适用于中文文本处理场景。

pypinyin 简介：
pypinyin 是一个用于将汉字转换为拼音的 Python 第三方库。它支持多种拼音风格（如普通拼音、首字母拼音等），并能够处理多音字、异体字等复杂情况。其简单易用且功能强大，广泛应用于中文自然语言处理、数据预处理等领域。
官方文档：https://pypi.org/project/pypinyin/
